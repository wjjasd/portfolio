### 2번. 호텔 서비스 플랫폼 (MyCONECT)

- **유형**: 호텔 종합 솔루션 앱
- **기여도**: 98%
- **기간**: 2021.01 ~ 2023.04
- **소속 회사**: 홀리츠

**기술 스택**
- 플랫폼: Android / Kotlin, Java
- 아키텍처: MVVM
- 비동기: RxKotlin
- 네트워크: Retrofit2 (REST API)
- 하드웨어: Android BLE (서드파티 SDK)
- UI: View Binding, Data Binding

**구성**
- 고객용 앱 (MyCONECT Guest): 호텔 검색, 예약, 체크인/아웃, 룸서비스, 결제, 발렛파킹
- BLE 제어: 도어락, 엘리베이터, 객실 기기 원격 제어

**작업 내역**
- 프로젝트 초기 설계부터 참여 (기여도 98%)
- 기능 개발, 디자인 리뉴얼, Google Play Store 배포, 성능 최적화, 유지보수 전 과정 담당
- REST API 기반 호텔 업무 처리 로직 구현
- BLE 기반 도어락·엘리베이터·객실 기기 원격 제어 구현 (서드파티 SDK 연동)
- MVVM + RxKotlin 아키텍처 적용

> 💡 **BLE 전문성 참고**: 사내 BLE 개발 도구로 Bluetena(Bluetooth 탐색·연결·Property Read/Write 앱) 별도 제작 — MyCONECT에서 축적한 BLE 역량 심화 목적

**성과**
- Google Play Store 출시 완료 (현재는 서비스 종료)
- 초기 설계부터 배포·유지보수까지 전 사이클 단독 주도
- REST API + BLE 멀티 인터페이스 통합 구현
- 정량 성과 수치: 미입력 (추후 작성)

**링크**
- 스토어 링크: 서비스 종료 (링크 없음)
- GitHub: 비공개 (회사 프로젝트)
- 증빙 자료: 핵심 코드 캡처 예정 (BLE 제어 로직, MVVM 구조 등 / 민감 정보 마스킹 후 활용)

### 7번. InOutCountBar (Android 오픈소스 라이브러리)

- **유형**: Android 오픈소스 커스텀 뷰 라이브러리
- **기여도**: 100%
- **기간**: 2025.12 ~ 2025.12
- **소속 회사**: 유타렉스 (사내 필요에서 출발 → 오픈소스 배포)

**기술 스택**
- 언어: Kotlin
- 플랫폼: Android Custom View
- 배포: JitPack

**기능 설명**
- 입장(IN), 퇴장(OUT), 현재 인원(COUNT) 세 가지 데이터 비율을 하나의 막대 그래프로 시각화
- 비율 변화에 따른 부드러운 애니메이션 제공
- 색상, 텍스트 크기 등 커스텀 속성 지원 (XML 설정 가능)
- JitPack을 통해 단일 의존성 추가로 바로 사용 가능

**성과**
- JitPack 배포 완료 (`com.github.wjjasd:InoutCountbar:1.0.1`) ✅
- GitHub 공개 레포지토리 운영 ✅
- 데모 GIF 제작 및 README 작성 완료 ✅

**링크**
- GitHub: https://github.com/wjjasd/InoutCountbar
- JitPack: https://jitpack.io/#wjjasd/InoutCountbar
