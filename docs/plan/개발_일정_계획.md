## 주차별 계획

```
Week 1: 환경 셋업 + 디자인 시안
─────────────────────────────────────
- 도메인/Vercel 셋업 (M1)
- Next.js + TypeScript + Tailwind 초기 구성 (M2)
- Google Stitch로 전체 섹션 시안 생성 (Hero, About, Projects, Contact)
- 마음에 드는 시안 선택 → Figma로 Export 후 디테일 조정
- Stitch에서 Tailwind CSS 코드 Export → 프로젝트에 적용 준비
- 콘텐츠 수집: 프로젝트 설명, 성과 수치, 자기소개 문구

Week 2: MVP 핵심 섹션 구현
─────────────────────────────────────
- Hero 섹션 (M3)
- About 섹션 (M4)
- Skills 섹션 (M6)

Week 3: Projects + Contact + 배포
─────────────────────────────────────
- Projects 섹션 (M5) ← 가장 중요, 시간 집중 투자
- Contact 섹션 (M7)
- 반응형 적용 (M8)
- SEO 설정 + 첫 배포 (M9, M10) ← MVP 완성!

Week 4: Should Have 구현 ①
─────────────────────────────────────
- Experience 타임라인 (S1)
- 프로젝트 상세 페이지 (S2)
- 이력서 PDF 다운로드 (S7)

Week 5: Should Have 구현 ②
─────────────────────────────────────
- 다크모드 (S3)
- Framer Motion 애니메이션 (S4)
- 문의 폼 EmailJS 연동 (S5)
- Google Analytics 연동 (S6)

Week 6: QA + 최종 점검 + 런칭
─────────────────────────────────────
- 전 페이지 QA (모바일/태블릿/데스크톱)
- Lighthouse 점수 측정 및 최적화
- 지인 피드백 수렴 및 수정
- LinkedIn, GitHub, 이력서에 URL 등록
```
