# 포트폴리오 웹사이트

프로덕트 엔지니어 양기정의 포트폴리오 웹사이트 레포지토리입니다.

---

## 레포지토리 구조

```
portfolio/
├── content/                       # 포트폴리오 콘텐츠 원본 (md + 미디어)
│   ├── 01_about/                  # 자기소개, 프로필 사진
│   ├── 02_projects/               # 프로젝트별 project.md + screenshots/
│   ├── 03_skills/                 # 기술 스택
│   ├── 04_experience/             # 경력
│   └── 05_design/                 # 아키텍처 설계 문서
└── docs/                          # 기획·계획·작업 문서
    └── plan/                      # 개발 일정 계획, 기획서
```

## 개발 계획

`docs/plan/개발_일정_계획.md` 참고 (6주 스프린트)

## 기술 스택 (예정)

- **프레임워크**: Next.js 14 (App Router)
- **언어**: TypeScript
- **스타일링**: Tailwind CSS
- **애니메이션**: Framer Motion
- **배포**: Vercel
