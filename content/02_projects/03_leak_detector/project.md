# 누수 감지 시스템

## 한 줄 요약

SubG 무선 통신 기반 누수 감지 펌웨어 커스터마이징 + Android 앱 단독 설계·구현, 납품 완료

## 기본 정보

| 항목 | 내용 |
|------|------|
| 유형 | 액체감지 센서, 경광등, 안드로이드 앱 프로젝트 |
| 기여도 | 80% |
| 기간 | 2025.12 ~ 2026.01 |
| 소속 | 유타렉스 |

## 배경 / 문제 정의

현장에서 누수를 실시간으로 감지하고 관리자에게 알리는 시스템이 필요했다. SubG(Sub-GHz) 무선 통신을 사용해 장거리 저전력 통신을 확보하고, Android 앱으로 감지 데이터를 시각화·관리하는 솔루션을 개발했다.

## 나의 역할과 기여

- SubG 펌웨어 샘플 소스 기반 커스터마이징
- Android 앱 설계 및 구현 (단독)
- 동료 분담: 빌드 환경 세팅, 리피터 기능 추가 납품

## 기술 스택

| 영역 | 기술 |
|------|------|
| 펌웨어 | SubG (Sub-GHz 무선 통신) |
| 앱 | Android Native / Kotlin |
