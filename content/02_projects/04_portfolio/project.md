## 배경 / 문제 정의

이직·취업 활동 및 개인 브랜딩을 위한 중앙 허브가 필요했다. 코드 작성 능력뿐 아니라 문제 정의→설계→구현→배포 전 사이클 경험과 비즈니스 임팩트를 직접 보여줄 수 있는 사이트를 목표로 설계했다.

## 나의 역할과 기여

- 기획·디자인·개발·배포 전 과정 단독 수행
- Claude AI를 활용한 기획·설계·코드 작성 (AI 협업 개발 워크플로우 적용)
- SSR/SSG 혼용으로 SEO 최적화 적용
- 반응형 레이아웃 (모바일 ~ 데스크톱) 구현
- Framer Motion 기반 페이지 전환·스크롤 애니메이션 구현
- Tistory RSS 파싱으로 블로그 최신 글 자동 연동
- Lighthouse 성능 90점 이상 달성 목표
