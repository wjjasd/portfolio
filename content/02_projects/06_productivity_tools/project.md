동료들이 실제로 쓰는 메모 앱과 반복 집계 업무를 자동화한 Python 스크립트 — 사내 실사용 검증 완료

| 기간 | 2026.04 |

## 프로젝트 개요

사내에서 반복적으로 발생하는 두 가지 문제를 직접 해결하기 위해 제작한 도구 모음. 첫째, 업무 메모·공유가 불편해 동료들이 직접 사용할 수 있는 메모 앱을 제작했다. 둘째, 제품 데이터 수집률 집계 작업이 수작업으로 반복되어 Python 기반 자동화 스크립트로 대체했다.

## 나의 역할과 기여

- 사내 메모 앱 기획·개발 전 과정 단독 수행 (동료 실사용)
- 데이터 수집률 집계 자동화 스크립트 설계·구현 단독 수행
- 반복 수작업을 코드로 대체해 업무 효율 개선

## 기술 스택

| 영역 | 기술 |
|------|------|
| 자동화 스크립트 | Python / openpyxl |
| 메모 앱 | Python / tkinter / winreg / threading |

## Daily Reporter 주요 기능

- 출근·퇴근 버튼으로 하루 시작·종료 기록 (날짜별 txt 파일 자동 저장)
- 매 정시마다 팝업 알림 — 현재 업무 내용 입력 유도 (미입력 4회 누적 시 자동 퇴근 처리)
- 마지막 기록 메인 화면 표시 및 클립보드 복사
- Windows 레지스트리 기반 시작프로그램 등록/해제 (설정 메뉴)
- PyInstaller로 단일 exe 배포 (아이콘 포함)
